# 🚗 Ford Car Price Predictor

## 📌 Project Overview

This project is a Streamlit web application developed during the AIML Internship.

The application predicts the estimated selling price of a Ford car using a trained Linear Regression model.

## ✨ Features

- User-friendly Streamlit interface
- Input car details
- Machine Learning prediction
- Linear Regression model
- One-Hot Encoding
- Feature Scaling

## 🛠 Technologies Used

- Python
- Streamlit
- Pandas
- Scikit-learn
- Joblib

## 📂 Project Structure


app.py
create_model.py
LR_model.pkl
scaler.pkl
columns.pkl
ford_car_dataset.xlsx
requirements.txt
README.md
